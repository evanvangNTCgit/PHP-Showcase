# Steps
- Clone/Download my repository
- Run the sampledb.sql within your phpmyadmin Database
- Drop my repository folder within the HTDOCS of XAMMP
- Within data context modify the connections strings accordingly to your environment
- Navigate to the index.php file within your local hosting of XAMMP
